Welcome to the Weather compare!

This is a site that was used to learn Django framework a bit more properly

It's main function is to compare weather forecasts from multiple websites. As a user without account you can access all places through "Places" at the navbar and then click on the individual places, created by other users, to see forecast for them.

You can also create an account which will allow you to create new places, for which the website will then show forecasts
